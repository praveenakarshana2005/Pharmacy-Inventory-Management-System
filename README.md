# 💊 Pharmacy Inventory Management System

## 📌 Overview
Pharmacy Inventory Management System is a desktop-based application developed using JavaFX.  
This system helps manage medicines, stock levels, sales, and inventory operations efficiently.

---

## 🚀 Features
- 📦 Manage Medicine Inventory
- ➕ Add / ✏️ Update / ❌ Delete Medicines
- 🔍 Search Medicines بسهولة
- 📊 Track Stock Levels
- 🧾 Sales Management
- ⚠️ Low Stock Alerts
- 🖥️ User-Friendly JavaFX Interface

---

## 🛠️ Technologies Used
- Java
- JavaFX
- MySQL (or any database you used)
- JDBC
